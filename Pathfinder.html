Help

Clicking one of the "add" buttons will allow you to click and drag across the grid to modify the nodes of the grid.

A start node is a point or points where the algorithm will start its search

An end node is a point or points where the algorithm will end its search

The algorithm will search for the shortest path from a start to an end node, multiple start and end nodes can be created

A wall node is an impassable obstacle that the algorithm will have to move around

A weight node adds to the cost of moving over it, normal nodes have a cost of 1

When the algorithm is finished a shortest, lowest cost path will be highlighted

If no path is possible, a path will not be highlighted
